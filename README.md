## 📌 معرفی پروژه

**APZSchool** یک اکوسیستم آموزشی و مشارکتی است که با هدف پرورش نیروهای متخصص در حوزه‌ی **بلاکچین**، **امنیت زیرساخت** و **توسعه‌ی نرم‌افزارهای غیرمتمرکز** طراحی شده است. ما با تکیه بر اصول **متن‌باز** و **شفافیت**، پلی می‌سازیم میان تئوری آکادمیک و نیازهای عملی صنعت.

اینجا، دانش‌آموزان، توسعه‌دهندگان و علاقه‌مندان می‌توانند در یک جامعه‌ی امن و مشارکتی، مهارت‌های خود را از پایه تا سطح پیشرفته ارتقا دهند و مستقیماً در پروژه‌های واقعی متن‌باز مشارکت کنند.

## ⚙️ معماری فنی و کیت توسعه (SDK)

در قلب APZSchool، یک کیت توسعه‌ی سبک و کارآمد (SDK) قرار دارد که به کارآموزان اجازه می‌دهد تا به‌صورت عملی با زنجیره‌ی بلوکی ارتباط برقرار کنند. نمونه کد زیر، نحوه‌ی اتصال به یک RPC و انجام تراکنش‌های پایه را نشان می‌دهد:

```typescript
import axios, { AxiosInstance } from 'axios';
import { BalanceResponse, Transaction } from './types';

export class APZClient {
  private rpc: AxiosInstance;

  constructor(rpcUrl: string) {
    this.rpc = axios.create({ baseURL: rpcUrl });
  }

  /**
   * دریافت موجودی یک آدرس خاص از شبکه
   */
  async getBalance(address: string): Promise<BalanceResponse> {
    const response = await this.rpc.get(`/account/${address}`);
    return response.data;
  }

  /**
   * ارسال تراکنش به شبکه (شامل امضا و سریال‌سازی)
   */
  async sendTransaction(tx: Transaction): Promise<string> {
    const response = await this.rpc.post('/tx', tx);
    return response.data.txHash;
  }
}
```

این کد بخشی از دوره‌ی «توسعه‌ی کلاینت‌های بلاکچین» است که دانش‌پذیران با استفاده از آن، نحوه‌ی تعامل با گره‌های شبکه و مدیریت هش‌ها را به‌صورت عملی می‌آموزند.

📚 سرفصل‌های آموزشی (مسیر یادگیری)

1. سطح مقدماتی (Fundamentals)
   · رمزنگاری کلید عمومی/خصوصی
   · ساختار بلوک و زنجیره
   · آشنایی با پروتکل‌های همسان‌سازی (Consensus)
2. سطح متوسط (Development)
   · پیاده‌سازی قراردادهای هوشمند با Solidity/Rust
   · کار با کتابخانه‌های Web3.js و Ethers.js
   · امنیت API و مدیریت کلیدها (HashiCorp Vault، AWS KMS)
3. سطح پیشرفته (Security & Infra)
   · تحلیل آسیب‌پذیری‌های قرارداد (Reentrancy، Overflow)
   · راه‌اندازی نودهای اختصاصی و معماری میکروسرویس
   · مهندسی زیرساخت‌های مقاوم در برابر ۵۱٪ Attack

ما در APZSchool به شفافیت پایبندیم:

· فرآیند بررسی کدها (Code Review) توسط متخصصین امنیت انجام می‌شود.
· جامعه‌ی ما دارای قوانین سختگیرانه‌ی اخلاقی و حریم خصوصی برای حفظ امنیت اعضاست.

🌐 جامعه و نحوه‌ی مشارکت
